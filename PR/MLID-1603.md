# [MLID-1603] fix(layout): prevent header from overlapping sidebar panel

## Summary

- Removed redundant `position: sticky`, `top: 0`, and `z-index: 100` from the DocumentIntake `.header` — these had no layout effect since the container already constrains height (`calc(100vh - 160px)`) with `overflow: hidden`, and scrolling is handled by the `.content` sibling
- Raised sidebar `z-index` from `100` to `1100` to prevent future stacking conflicts with other components in the 100–1000 range
