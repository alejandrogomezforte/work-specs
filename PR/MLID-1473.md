# [MLID-1473] Dashboard - Expand tables to full width

## Summary

This PR updates the central UI frame to a full-width layout across the entire application. The main changes include:

- Removed all `max-width` constraints from inner containers
- Removed horizontal padding from inner containers (side padding now comes only from the main layout)
- Main layout container now provides consistent **24px side margins** globally
- All inner containers now use **vertical-only padding (`20px 0`)**
- Removed white background from appointments container for visual consistency
